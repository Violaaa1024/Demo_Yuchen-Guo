 Demo for AI-assisted invoice extraction in enterprise scenarios
 
 发票信息自动提取插件（Invoice Extractor）

一个基于规则匹配的浏览器插件，用于自动提取PDF发票中的关键信息，并辅助完成录入流程，减少人工操作成本。

⸻
如何在浏览器中安装

适用于 Chrome / Edge 浏览器

	1.	下载或克隆本项目到本地，并解压文件夹
	2.	打开浏览器，进入扩展管理页面：
	•	Chrome：chrome://extensions/
	•	Edge：edge://extensions/
	3.	打开右上角 “开发者模式（Developer Mode）”
	4.	点击 “加载已解压的扩展程序（Load unpacked）”
	5.	选择本项目解压后的文件夹
	6.	插件加载完成，即可在浏览器中使用

⸻

功能说明
	•	📄 PDF发票解析：支持读取PDF格式发票内容
	•	🔍 关键字段提取：自动识别发票金额、日期、发票号等核心信息
	•	⚙️ 多策略匹配机制：结合精准匹配、模糊匹配及兜底规则，提高识别准确率
	•	🧩 自动填充能力：提取结果可用于自动填充表单或导出结构化数据

⸻

设计思路

针对企业在发票处理过程中依赖人工录入、效率低且易出错的问题，本插件通过规则匹配与文本修复逻辑，对PDF解析中的常见问题（如字段断裂、格式不统一）进行优化处理，在保证准确性的同时显著提升处理效率。

⸻

适用场景
	•	财务发票录入
	•	运营数据整理
	•	企业内部ERP流程辅助

⸻

⚠️ 注意事项
	•	当前版本主要基于规则匹配，对格式差异较大的发票适配有限
	•	建议在标准发票模板下使用，以获得更稳定效果

⸻

后续优化方向

	•	支持更多票据类型（如电子收据、合同等）
	•	增强异常字段识别与纠错能力
